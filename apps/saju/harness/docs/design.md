# Saju Design

## 1. Purpose
이 문서는 `saju` 앱의 화면 인상과 정보 배치 기준을 정의한다.
목표는 멋을 과하게 만드는 것이 아니라, 입력부터 결과까지 이해가 빠른 화면을 만드는 것이다.

## 2. Chosen Direction
- 조용하고 밝은 교육용 화면
- 입력과 결과가 분리된 단순 흐름
- 오행 정보는 색과 텍스트를 함께 써서 보여주기

## 3. Layout Rule
- 첫 화면에서는 입력 영역이 먼저 보여야 한다.
- 결과 화면에서는 팔자, 오행, 해석 순서로 보여야 한다.
- 푸터 안내는 입력 전과 결과 후 모두 같은 위치에 둔다.

## 4. Visual Rule
- 기본 화면은 복잡한 장식 없이 간결하게 유지한다.
- 오행 정보는 일관된 색 체계를 쓴다.
- 색만으로 의미를 전달하지 않고 이름을 함께 적는다.
- 읽기 어려운 커스텀 효과보다 기본 컴포넌트를 우선한다.

## 5. Five Elements Colors
- 목: `#A8D5BA`
- 화: `#F4A9A8`
- 토: `#F5D89A`
- 금: `#E8D5A8`
- 수: `#A8C8E8`
## 6. Responsive Rule
- 모바일에서도 입력과 결과 순서가 깨지지 않아야 한다.
- 입력 필드는 모바일 표준 화면에서도 한 번에 고를 수 있어야 한다.
- 결과 카드는 모바일 표준 화면에서 자연스럽게 줄바꿈되어야 한다.

## 7. Out of Scope
- 과한 애니메이션
- 이미지 에셋 중심 화면
- 설명 없는 장식 요소
