# Saju Product Sense

## 1. Purpose
`saju`는 생년월일시를 입력하면 사주 팔자와 오행 분포를 보여주는 교육용 앱이다.
사주는 흥미를 끄는 소재이고, 핵심 목적은 초보자가 harness engineering으로 앱 만들기를 구현해보게 만드는 것이다.

## 2. Target User
- harness engineering 방식으로 앱을 만들어 보고 싶은 사람
- 사주라는 친숙한 주제로 작은 앱을 끝까지 완성해 보고 싶은 사람
- 복잡한 서비스보다 작은 교육용 앱을 원하는 사람

## 3. Product Rule
- 정확도를 우선한다.
- 설명 가능한 규칙 기반 결과만 보여준다.
- 초보자가 읽기 어려운 복잡성은 줄이거나 범위를 축소한다.

## 4. In Scope
- 생년월일시 입력
- 사주 팔자 표시
- 오행 분포 표시
- 짧은 규칙 기반 해석

## 5. Out of Scope
- 운세 예측
- 궁합 비교
- 사용자 데이터 저장
- 설명 없는 AI 생성 해석

## 6. Success
- 초보자가 harness 문서와 구현 흐름을 읽고 따라갈 수 있다.
- 핵심 결과가 문서와 테스트로 검증된다.
- 앱 실행과 검증 과정이 과하지 않다.

## 7. Failure
- 기능이 늘면서 교육용 구조가 깨진다.
- 검증할 수 없는 해석이나 동작이 들어간다.
- 사용자 입력 데이터가 남거나 새어 나간다.
