# Completed Plans

## Purpose
이 폴더는 완료된 작업 문서를 보관한다.
완료된 문서는 과거 결정과 실패 이력을 추적할 때 다시 본다.

## Move Rule
- 마지막 verdict가 `APPROVED`일 때만 이 폴더로 옮긴다.
- 옮기기 전 마지막 state와 next step을 정리한다.
- 완료 후에는 같은 문서를 다시 진행 중 상태로 바꾸지 않는다.

## Record Rule
- 최종 산출물이 무엇인지 남긴다.
- 남겨 둔 위험이나 보류 항목이 있으면 적는다.
- 같은 root_cause로 반복된 이력이 있으면 유지한다.
