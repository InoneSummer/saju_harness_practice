# Saju Product Spec

## 1. Purpose
이 문서는 `saju` 앱의 입력, 출력, 핵심 동작을 정의한다.
구현과 테스트는 이 문서를 기준으로 맞춰야 한다.

## 2. Input
- 년: 1900~2050 정수 선택
- 월: 1~12 선택
- 일: 1~31 선택
- 시: 자시~해시의 12개 시간 구간 중 하나 선택

## 3. Output
- 사주 팔자
- 오행 분포
- 규칙 기반 해석 텍스트
- 입력 오류 시 일반 오류 메시지

## 4. Core Flow
1. 사용자가 년, 월, 일, 시를 선택한다.
2. 앱이 입력 유효성을 확인한다.
3. 유효하면 사주 팔자와 오행 분포를 계산한다.
4. 결과를 화면에 표시한다.
5. 유효하지 않으면 오류 메시지를 표시한다.

## 5. Validation Rule
- 빠진 입력은 실패로 처리한다.
- 존재하지 않는 날짜는 실패로 처리한다.
- 지원 범위 밖 입력은 실패로 처리한다.
- 지원 범위는 1900-01-01부터 2050-12-31까지로 본다.

## 6. Domain Rule
- 결과는 설명 가능한 규칙 기반 계산이어야 한다.
- 년주, 월주, 일주는 기준 라이브러리와 맞아야 한다.
- 시주는 내부 조견표 기준을 따라야 한다.
- 오행 개수 합계는 항상 8이어야 한다.

## 7. Out of Scope
- 운세 예측
- 궁합 비교
- 데이터 저장
- 자유 생성형 해석
